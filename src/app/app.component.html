2 准备翻译组件

要准备翻译项目，请完成以下操作。

  • 使用 i18n 属性标记组件模板中的文本
  • 使用 i18n- 属性在组件模板中标记属性文本字符串
  • 使用带 $localize 标记的消息字符串标记组件代码中的文本字符串

2.1 在组件模板中标记文本

语法：{{ p1 }}

i18n_metadata= meaning|description@@custom_id

meaning: 提供文本在特定上下文中的含义或意图

description: 提供额外的信息或背景

custom_id: 提供自定义标识符

示例：

你好~

2.2 标记翻译的元素属性

语法：{{ p2 }}

示例：

这是元素属性示例

2.3 在组件代码中标记文本

语法：{{ p3 }}

示例：

{{ str }}

2.4 包含插值的文本

模板中插值: {{p4}}

示例:

我是{{ name }}

{{str1}}
